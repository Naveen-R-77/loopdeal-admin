# LoopDeal Admin Application: Reports & Analytics

## 1. Dashboard Overview
The main dashboard provides real-time visibility into the store's performance. It tracks four primary metrics:
- **Total Users**: The number of registered administrators and customers.
- **Total Products**: Live inventory count across all locations using the `LoopDeal` catalog.
- **Total Orders**: Total transaction volume processed through the hub.
- **Revenue**: Gross revenue in INR (₹) derived from successful transactions.

## 2. Sales Analytics (Reports)
- **Monthly Revenue Graph**: Displays sales trends over a 6-month period (Oct to Mar). This allows admins to identify seasonal peaks and plan inventory accordingly.
- **Category Sales**: In-depth breakdown of which technology segments (CCTV, Network, Computing) are generating the most revenue.

## 3. Inventory Distribution (Live Tracking)
- **Automatic Sync**: The "Live Inventory Distribution" pie chart is dynamically calculated. It scans the current product list and categorizes it in real-time.
- **Categories Tracked**: 
  - **CCTV Cameras**: Surveillance hardware and accessories.
  - **NVR & DVR**: Recording and storage infrastructure.
  - **Network**: Routers, switches, and enterprise connectivity.
  - **Desktops**: Enterprise computing and hardware.
- **Dynamic Updates**: When a new product is added to a category, the pie chart automatically updates its percentage distribution.

## 4. Operational Logs
- **Activity Log**: Accessible from the notification menu, this log tracks every sign-in, product update, and order modification for security auditing.
- **Delayed Order Monitor**: Orders with status "Delayed" are flagged for immediate administrative review to ensure customer satisfaction.
